# Plugin Assets

This directory should contain the following WordPress.org plugin assets:

## Required Images (for WordPress.org submission)

1. **icon-128x128.png** - Plugin icon (128x128 pixels)
2. **icon-256x256.png** - Plugin icon (256x256 pixels, optional but recommended)
3. **banner-772x250.png** - Plugin banner (772x250 pixels)
4. **banner-1544x500.png** - Plugin banner for high-DPI displays (1544x500 pixels, optional)

## Screenshot Images (for WordPress.org)

- **screenshot-1.png** - Plugin settings page with theme color picker
- **screenshot-2.png** - Social media configuration form
- **screenshot-3.png** - Chatbot appearance on frontend
- **screenshot-4.png** - Mobile-responsive chatbot interface

## Design Guidelines

- Use ChatMaxima brand colors and logo
- Keep images professional and clear
- Screenshots should show the plugin in action
- Icons should be simple and recognizable at small sizes

## File Formats

- All images should be PNG format
- Use high-quality images (minimal compression)
- Ensure proper dimensions for each asset type

These assets will be uploaded separately to the WordPress.org SVN repository in the `/assets` folder.
